🚀 NPM DEPLOYMENT - ENVIRONMENT VARIABLES NEEDED
════════════════════════════════════════════════════════════════════════════════

FOR PUBLISHING TO NPM:

1. LOCAL DEPLOYMENT:
   ├─ Run: npm login
   ├─ Enter your npm credentials
   ├─ Then: pnpm publish:patch (or :minor, :major)
   └─ Stored in: ~/.npmrc (DO NOT COMMIT)

2. CI/CD DEPLOYMENT (GitHub Actions):
   ├─ NPM_TOKEN (get from https://www.npmjs.com/settings/~/tokens)
   ├─ Add to: GitHub > Settings > Secrets > New secret
   ├─ Name: NPM_TOKEN
   ├─ Value: (paste your automation token)
   └─ Workflow will auto-publish on git tag push

FOR RUNNING THE SERVER:

1. AUTHENTICATION (Choose one):
   ├─ BITBUCKET_TOKEN=your_app_password (RECOMMENDED)
   └─ BITBUCKET_USERNAME=email + BITBUCKET_PASSWORD=password

   Get from: https://bitbucket.org/account/settings/app-passwords/
   Permissions needed:
     ✓ Pipelines: Read
     ✓ Repositories: Read, Write
     ✓ Pull requests: Read, Write

2. CONFIGURATION (Optional):
   ├─ BITBUCKET_WORKSPACE=my-workspace
   ├─ BITBUCKET_URL=https://api.bitbucket.org/2.0
   └─ BITBUCKET_ALLOW_DANGEROUS=true (only in dev)

3. LOGGING (Optional):
   ├─ BITBUCKET_LOG_DISABLE=false (enable logging)
   ├─ BITBUCKET_LOG_DIR=/custom/path
   └─ BITBUCKET_LOG_PER_CWD=true

📝 QUICK DEPLOYMENT GUIDE
════════════════════════════════════════════════════════════════════════════════

STEP 1: Setup NPM Credentials
────────────────────────────
Option A (Local):
    npm login

Option B (CI/CD):
    1. Go to: https://www.npmjs.com/settings/~/tokens
    2. Create "Automation" token
    3. Add to GitHub Secrets as: NPM_TOKEN

STEP 2: Verify Everything Works
──────────────────────────────
    pnpm build && pnpm lint && pnpm test

STEP 3: Publish to NPM
────────────────────
LOCAL:
    pnpm publish:patch   # v5.0.6 → v5.0.7 (bug fixes)
    pnpm publish:minor   # v5.0.6 → v5.1.0 (new features)
    pnpm publish:major   # v5.0.6 → v6.0.0 (breaking changes)

CI/CD:
    git tag v5.0.7
    git push origin v5.0.7

STEP 4: Verify Publication
──────────────────────────
    • Check: https://www.npmjs.com/package/bitbucket-mcp
    • Test: npm install -g bitbucket-mcp
    • Run: bitbucket-mcp (with env vars set)
